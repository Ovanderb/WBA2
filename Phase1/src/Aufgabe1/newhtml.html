Jede XML Datei benötigt als erstes eine XML Deklaration <?xml version="1.0" ?>. Hiermit wird festgelegt, um welche XML Version es sich bei dem verwendeten Dokument handelt. Weitere Parameter, z.B. das Encoding Encoding="UTF-8", sind optional. Danach kann optional eine DTD ( Document Type Definition ) in die Datei eingebunden werden. Diese kann entweder als eine externe Datei eingebunden oder direkt in der Datei definiert werden. Die Empfehlung ist, eine DTD immer einzubinden, denn ohne DTD können XML Dateien zwar wohlgeformt sein, aber sie können nicht gegen eine DTD validiert werden.
